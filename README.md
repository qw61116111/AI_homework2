---------------------預測手法----------------------
我是用前十四天來預測下一天股票，並且依照預測出來的值判斷是否漲跌

假設原本我有一支股票，然後我預測出明天會漲，那麼下一天就輸出-1

如果預測明天會跌，那就輸出0，直到股票漲則賣出

反之賣空股票則相反

另外若我手上沒有持股的話，則會random(買,賣空)來決定初始狀態，再重複上面的循環

algo概念如下

  if (沒有持股):

    staus=random(買,賣空)

  esle: //有持股

    if staus==買:

      predict(使用前14天資料)

      if(predict漲)

        賣出

      else:

        hold

    else: //staus==賣

      predict(使用前14天資料)

      if(predict跌)

        買入

      else:

        賣
